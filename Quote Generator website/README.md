# Quote of the Day App

A simple and clean **Quote of the Day** web application built using **HTML, CSS, and JavaScript**.  
It fetches random quotes from an API and allows users to share quotes on Twitter.

## Features
- Fetches random quotes using API
- Displays quote and author
- New Quote button to load another quote
- Tweet button to share quote on Twitter
- Clean and simple UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Quotable API

## How to Use
1. Open the project in your browser
2. A quote loads automatically on page load
3. Click **New Quote** to generate another quote
4. Click **Tweet** to share the quote on Twitter

## API Used
- https://api.quotable.io/random
