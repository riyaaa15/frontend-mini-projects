# OTP Generator & Verification

## Overview
A simple OTP generator and verification app built with **HTML, CSS, and JavaScript**.  
Users can generate a 6-digit OTP, copy it, and verify it within a set time limit.

## Features
- Generate **6-digit OTP**  
- OTP displayed in a **read-only button**  
- **Countdown timer** (21 seconds)  
- Button disabled during countdown  
- **Copy OTP** to clipboard  
- OTP verification with messages:  
  - ✅ Correct OTP  
  - ⚠️ Wrong OTP  
  - ⏰ OTP expired  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## How to Use
1. Click **Generate OTP**  
2. Copy OTP using **Copy OTP** button (optional)  
3. Enter OTP in the input field  
4. Click **Verify** to check OTP  
5. See the message for success, error, or expiry  

## UI Highlights
- Rounded container with shadow  
- Clean buttons with hover and disabled states  
- OTP display shows **not-allowed cursor**  
- Responsive layout for desktop and mobile
